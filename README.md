# LOTG
## Library On The Go

### Features of LOTG:
  * Hosted Locally
  * Easy Interface for both Admin and Student
  * Easy addition of new Data
  * Features like: Book Availability, Books Left, Book Request, Book Review/Comments

  * Tested on Win 7, Win 10, Centos 7, Ubuntu 16.
  
### Languages used:
  * PHP
  * SQL
  * HTML
  * CSS
  * Javascript
  
### Softwares used:
  * XAMPP with MySQL
