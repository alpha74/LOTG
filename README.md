LIBRARY ON THE GO
======================

# FEATURES:
  * Hosted Locally
  * Easy Interface for both Admin and Student
  * Easy addition of new Data
  * Features like: Book Availability, Books Left, Book Request, Book Review

  * Tested on Win 7, Win 10, Centos 7, Ubuntu 16.
  
# LANGUAGES USED:
  * PHP
  * SQL
  * HTML
  * CSS
  * Javascript
  
# SOFTWARES USED:
  * XAMPP with MySQL
