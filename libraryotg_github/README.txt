Library Management System
==========================
Name: LOTG(Library On the Go)
============================


# FEATURES:
  1. Hosted Locally
  2. Easy Interface for both Admin and Student
  3. Easy addition of new Data
  4. Features like: Book Availability, Books Left, Book Request, Book Review
  
  
# LANGUAGES USED:
  1. PHP
  2. SQL
  3. HTML
  4. CSS
  5. Javascript
  
# SOFTWARES USED:
  1. XAMPP with MySQL
